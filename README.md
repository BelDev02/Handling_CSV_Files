# Handling_CSV_Files-# CSV File Handling in Python

Ce projet montre comment créer, lire et afficher des fichiers CSV en Python en utilisant le module standard `csv`. Il inclut des exemples pour :

- Écrire des données dans un fichier CSV
- Lire les données sous forme de dictionnaire
- Afficher les données sous forme de tableau

## Contenu du projet

- `File_handling.py` : script manipule le fichier CSV
- `Files.csv` : fichier CSV généré par le script
- `README.md` : ce fichier de documentation

## Prérequis

- Python 3.x installé
- Aucun module externe requis
